# SEO Baseline

## Meta Tags
- **Title Pattern**: Defined in layouts (need to inspect)
- **Description**: Sourced from `excerpt` or `description` frontmatter.

## URL Structure
- **Posts**: defined by `permalink` frontmatter. Example: `/editorial/bienvenida/`
- **Archives**: `/tags/[tag]/`, `/categories/[category]/` (inferred)
- **Trailing Slash**: Appears to be enforced (folders in `dist`).

## Sitemap
- **Generated by**: `@astrojs/sitemap` (inferred from `sitemap-index.xml`)
- **Location**: `sitemap-index.xml` pointing to `sitemap-0.xml`

## Feed
- **RSS**: `rss.xml` exists in root.
