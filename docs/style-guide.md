# Noticiencias Style Guide

## Tipografia
- Titulares: `Sora` (600/700)
- Texto: `Source Serif 4` (300/400)
- UI y metadatos: `Sora` (400/500)

## Paleta de color
- Ink principal: `#1c1f24`
- Ink secundario: `#5a636d`
- Accent ciencia: `#0f6f63`
- Accent oscuro: `#0b4f47`
- Fondo papel: `#f4f2ed`
- Borde suave: `#e3e0d8`
- Aviso/alarma: `#c7522a`

## Espaciado y grid
- Max width editorial: 1200px
- Secciones principales: 48px arriba/abajo
- Cards: radius 16px, padding 20-24px

## Componentes clave
- Hero editorial: manifiesto, CTA doble, indicadores de rigor.
- Cards de investigacion: imagen, kicker, titulo, metadatos.
- Ficha editorial: metodo de traduccion, score, revision, confianza.
- Bloque "Por que importa": callout con borde de acento.

## Imagenes
- Hero: 16:9 o 3:2, contraste alto.
- Cards: 16:9 con foco en textura o laboratorio.
- Siempre declarar `image_alt` en front matter.
